# アイコンファイルについて

現在、アイコンファイル（icon16.png、icon48.png、icon128.png）が必要です。

## アイコンの作成方法

以下のいずれかの方法でアイコンを作成してください：

1. **オンラインツールを使用**
   - https://www.favicon-generator.org/ などでPNG画像を生成
   - 16x16、48x48、128x128の3サイズを用意

2. **既存の画像から変換**
   - マイクやスピーカーのアイコン画像を用意
   - 各サイズにリサイズ

3. **一時的なプレースホルダー**
   - 開発中は、任意の16x16、48x48、128x128のPNG画像を配置してください
   - ファイル名: icon16.png、icon48.png、icon128.png

## 一時的な回避策

開発中は、Chrome拡張機能の読み込み時にアイコンがなくてもエラーになりません。
manifest.jsonのiconsセクションをコメントアウトすることで、一時的に回避できます。
